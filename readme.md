# iOS10-SocialNetWork

Creation of a simple Social App

### Concepts Used:
* Firebase:
  * Authentication
  * Database
    * Observers
  * Storage
* FacebookLogin sdk
* SwiftKeychainWrapper
* UITableView
* UIGestureRecognizer
* UIImagePickerView
  * Edit an image
* NSCache for images
* Creation of Delegates
* Closures
* NotificationCenter (responsive layout)
  * UIKeyboardWillHide
  * UIKeyoardWillShow
